## Run Locally

Clone the project

```bash
  git clone https://github.com/drabiot/ft_vox.git
```

Go to the project directory

```bash
  cd ft_vox
```

Compile the project

```bash
  make -j
````

Open the project

```bash
  ./ft_vox
```
